# VX Web V2: Spatial Experience Platform Vision

## 1. 프로젝트 개요 (Project Overview)
**VX Web V2**는 단순한 행사 안내 웹사이트를 넘어, 대규모 국제대회 및 컨벤션의 물리적 공간과 디지털 정보를 실시간으로 결합하는 **공간 경험 플랫폼(Spatial Experience Platform)**입니다.

본 프로젝트는 "사용자가 정보를 찾아다니는 것이 아니라, 공간이 상황에 맞춰 정보를 제안한다"는 **Runtime UX** 철학을 바탕으로 설계되었습니다. 하네스 엔지니어링(Harness Engineering) 기법을 도입하여, 실제 운영 환경에서 발생할 수 있는 혼잡도, 긴급 상황, 다국어 요구사항을 가상으로 시뮬레이션하고 검증할 수 있는 견고한 구조를 갖추고 있습니다.

## 2. 왜 필요한가? (The Problem & Solution)

### 기존 국제대회 운영의 문제점
*   **실내 길찾기 어려움**: 복잡한 대형 전시장 내에서 2D 지도만으로는 현재 위치와 목적지를 연결하기 어렵습니다.
*   **혼잡 관리 한계**: 특정 구역에 인파가 몰릴 경우, 이를 실시간으로 감지하고 분산을 유도할 수 있는 수단이 부족합니다.
*   **다국어 UX 부재**: 글로벌 참가자들을 위한 실시간 정보 번역 및 현지화된 운영 가이드가 미흡합니다.
*   **참가자 참여 저조**: 단순히 정보를 확인하는 수준에 그쳐, 행사장 전체를 탐험하고 경험하게 만드는 동기가 부족합니다.

### VX Web V2의 해결 방향
*   **Spatial Runtime UX**: 현재 시간과 위치, 행사장 상태를 분석하여 최적의 Hero 메시지와 내비게이션을 제공합니다.
*   **Crowd-aware Experience**: 실시간 혼잡도 데이터를 기반으로 우회 경로를 추천하고 최적의 출입구를 안내합니다.
*   **Immersive Exploration**: 360도 공간 뷰어와 인터랙티브 히트맵을 통해 방문 전 공간을 미리 체험하고 이해할 수 있게 합니다.
*   **Gamified Journey**: QR 기반의 스탬프 투어와 퀘스트 시스템을 통해 단순 방문객을 능동적인 탐험가로 전환시킵니다.

## 4. Spatial UX 방향성 (Spatial UX Philosophy)
VX Web V2는 **"생각하지 않는 내비게이션(Zero-thought Navigation)"**을 지향합니다.
*   **Contextual Awareness**: 사용자가 로비에 있으면 '입장'을, 세션 중에는 '진행률'을, 종료 직전에는 '다음 장소'를 최우선으로 노출합니다.
*   **Accessibility First**: 고령자나 장애인을 위해 별도의 '접근성 모드'를 지원하며, 폰트 크기와 대비, 이동 경로를 이에 최적화합니다.
*   **Emotional Connection**: 단순한 텍스트가 아닌 360도 이미지와 부드러운 애니메이션을 통해 공간에 대한 심리적 거리감을 좁립입니다.

## 5. 국제대회 운영 관점의 가치 (Operational Value)
*   **참가자 경험 향상**: 스트레스 없는 이동과 몰입감 있는 공간 탐험을 제공합니다.
*   **운영 효율 증가**: 실시간 공지와 자동화된 경로 안내로 운영 인력의 부담을 줄입니다.
*   **안전 관리 강화**: 긴급 상황 발생 시 즉각적인 대피 경로 안내 및 Red Alert 시스템으로 리스크를 최소화합니다.
*   **데이터 기반 최적화**: 향후 축적될 혼잡도 및 이동 데이터를 통해 행사 기획의 정밀도를 높일 수 있습니다.

---
**VX Web V2**는 미래형 컨벤션 운영의 표준을 제시하며, 전 세계 참가자들에게 가장 진보된 공간 경험을 선사할 준비가 되어 있습니다.
